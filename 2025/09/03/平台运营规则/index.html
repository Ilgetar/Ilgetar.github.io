# 社区规则

# 虚构体验、夸大其词、低质创作、笔记过度 P 图、素材搬运、诱导行为、导流行为、拉踩行为，这八点都将纳入重点监测范围。

案例：笔记使用 “当妈才知道”、“我女儿” 这类拟人的视角去介绍使用产品后的体验，在专业号笔记规则中是不允许的。

正确做法：如果是专业号复制转发了素人或达人的笔记内容，必须在文中 @原作者。

# 虚构体验 - 专业号违规转载 / 伪造体验

案例：笔记中存在非自身经历的伪造分享，出现笔记内容与账号本人身份不相符的情况，使可信度大打折扣。

# 夸大其词

案例：封面或标题中刻意使用夸张失真、博人眼球、价值观导向不良的表述进行产品推荐。

** 类型一:** 不良导向笔记标题或封面存在过度取悦异性、贬低女性友情等价值观问题，制造噱头，吸引眼球。

** 类型二:** 故弄玄虚 笔记采用 “千万不要买”、“买了后悔”、“我后悔买” 等标题，让人误认为良心避雷贴，但点击后才发觉是种草文而非吐槽的本质。

# 笔记过度 P 图 - 不合理对比暗示

** 案例:** 为凸显产品效果，用诸如几年前几年后、妆前妆后、与他人做对比等方式暗示产品功效，但非产品真实功效，有偷换概念、混淆试听的意图，常见于美白、抗老等合作产品效果展示。这种只要不太过分，一般不会被限流，但如果被人工判定违规基本不接受捞回，自行把握度。

# 虚构体验 - 图文直发

** 案例:** 博主直接使用品牌方生产的内容，伪装成自身真实消费体验进行图文直发的行为。该行为属于虚假种草，存在欺骗诱导用户的嫌疑。

** 案例:** 封面几篇不同博主笔记出现相似或相同的人脸图。

# 导流行为 - 二维码展示

** 案例:** 在笔记图片中露出二维码，明示或暗示用户可通过扫码进入其他平台进行浏览或关注。

# 导流行为 - 竞品导流

** 案例:** 通过评论区或私信回复，“淘宝、天猫旗舰店” 及其变体字形式，将用户导流至其他平台。竞品范围包括但不限于淘宝、京东、得物等电商平台，B 站、微博、抖音等社交媒体平台。

用谐音字表情符合去引导，不要过分明显，评论区不要回复一模一样的内容，不要提及竞品平台的名字。～

# 拉踩行为 - 通过刻意攀附 / 拉踩 / 贬低，大牌带小牌凸显特定品牌

** 案例:** 将不同品牌产品放在一起对比，模糊描述其他品牌的货源，并刻意突出自身的货源更加优质，来抬高自己。突出自己的成分，却给其他品牌打出负面评价。

自己的账户内容中少提及同行竞品的名称，可以模糊处理，比如这一行经常有什么问题，我们为了避免做了哪些优化，是我们的竞争力。

# 诱导行为 - 利益诱导

** 案例:** 通过抽奖送礼的方式，来诱导用户进行点赞、评论、收藏、分享、发文、关注等行为。违规内容除直接要求用户点赞 + 评论 + 收藏等，还包括 “在评论区抽取 XX 位幸运用户” 等变相诱导用户评论的行为。

只有引导私信的不算违规！

# 3. 时间管理

这些时间段小红书在线用户多，平台的流量自然也就多。但同时要注意这些时间段也是作品发布的高峰期，可以根据自己作品的内容和质量去选择最为合适的时间段。发布时间除了会影响平台的流量推荐，一定程度上也会决定用户阅读你笔记时的心情和状态。选择合适的发布时间，可以提高你笔记成为爆款的概率。

早班高峰 6—8 点：用户乘地铁、公交或步行中有大量的空闲时间，但因为时间零散不好做其他事情，所以通常会在社交平台上打发时间。这时候在线用户较多，而且是一天的开始，大部分人都保持着一个较好的心情状态，对内容更容易接受。

午休高峰 11—13 点：午饭过后刷刷手机是一个比较普遍的习惯。这个时间段同样是用户较活跃的时段，且用户的状态比较轻松，阅读的兴趣和耐心都较高。

下班高峰 16—18 点：这个时段大多数人都在下班途中，和早班高峰期一样，用户的空闲时间较多，会习惯性地打开社交软件。

睡前高峰 20 点 — 次日 1 点：晚饭过后是大部分人最闲适的时候，一般在晚 8 点左右，这时，用户活跃度高、在线时间长，且搜索内容的主动性强。

但是小红书运营的关键，并不在于黄金时间段发布，而在于以下三点：

运营逻辑：

平台给用户账号和博主内容都打上标签，大数据算法分辨博主发布的内容是什么，适合哪些人群，并做小范围的推荐测试。互动率有了（点赞、收藏、评论、转发等行为），再进行二次推荐。

平台用户：

通过用户的搜索、浏览、藏评藏等行为，被平台记录并打上对应标签，这就是信息茧房，平台摸清你的喜好后，推荐你想看的个性化内容。

博主内容：

笔记内容优质，不仅内容要垂直化，封面图、标题和正文也要合理布局关键词。

用户关注你，是在期待你在这块领域，能持续分享优质内容。是金子总会发光的。

# 5. 内容与平台定位

# 平台定位

小红书对于自己的定义是 “温暖的小社区”，社区的主流氛围是：珍惜自己当下的生活，并且为之付出努力。我们可以看到，哪怕许多 “年纪轻轻就躺平” 的、“97 年生人早早退休” 的，看起来是一种颓废，但其实仍然是对于生活的积极态度，想要打破现状、找到更好地自我的一种努力。

所以，在小红书，更受欢迎的分享内容是温暖、是积极、是分享细枝末节的美好的东西，而不是矛盾冲突、不是负面情绪，这也是创作者在小红书进行内容创作时需要注意的。

# 内容定位

小红书上的账号基本分两类，品牌号和个人号。其中，品牌号的内容大多会围绕产品本身或产品衍生内容来创作，再加以创意表达形式即可。如钟薛高的雪糕棍、麦当劳的纸袋笔筒、海底捞的美甲、联通客服的变装秀等等

内容的关键词布局，在标题、开头、话题、封面文字中使用垂直细分的关键词；注意优化种草内容的带货数据指标；内容更加生活化、真人感、有具体细节。

# 6. 平台算法

# 平台的推荐方法

平台算法推荐：根据用户搜索的关键词，搜索相关的内容给用户。

阅读衍生推荐：当用户主动浏览内容后，平台会推荐更多的内容给用户。

地域推荐：当点击某一地域时，平台会推荐相关的内容。

编辑推荐：小红书官方的推荐。

小红书的推荐机制和其他平台差不多，唯一的区别就是在推荐之前，它有一个收录的环节，如果内容未被收录就到不了推荐池，而检测内容是否被收录的方法就是搜索该内容，如果能搜索到，那就说明已收录，否则反之。

内容被收录后就会进入系统推荐池，首先内容会被推荐到一级流量池，大概有 200 曝光左右，如果内容的点击率、点赞率、收藏量、评论量等数据较好，达到了进入下一级流量池的条件，系统会自动将内容推荐到下一级流量池，大概有 2000 左右的曝光，以此类推，如果内容的数据达不到进入下一级的条件，那么就会被停止推荐。

在推荐期间，如果内容被举报，推荐量则会下降，如果你的一篇之前很火的内容突然就不涨阅读了，那很有可能就是被人举报了。在运营小红书时，切记让自己的账号增加权重，权重越高被推荐的几率也就越大，同时记得不要轻易违规，一旦违规，轻则降权重，重则账号报废。

# 私域运营

新媒体运营的尽头是私域。那么私域运营的尽头则是一个小公式：工具 + 福利 + 内容。

工具指的是平台内外的社群运营工具，我们即可以在平台内的社群里运营用户，也可以在平台外的其他社交软件内去运营用户，具体用哪个，根据你的产品和运营目的去选择即可。

福利指的是要有周期的给予用户一些有利的小活动，让用户与你互动，拉进彼此之间的距离，构建信任，促使用户裂变。

内容就不用多说了，每天在社群内发什么内容？什么内容能让用户持续关注？这需要在实际运营当中做深入的调研，不断改进内容，才能留住用户。

# 视频号审核机制

任何平台都有设定视频审核过滤机制，目前都是有双重审核，即机器审核和人工审核，视频号也是同样。

机器审核：

提前设置好人工智能模型来识别你的视频画面和关键词，用来审核你的作品和文案是否存在违规行为。

如果视频规则可能会导致作品被：降低推荐、删除作品、自我可见等。

严重违规则直接删除或者设置自我可见。如果无法直接判定违规则会进入人工审核。

人工审核：

对机器筛选出来疑似违规的作品，由人工进行审核。人工审核主要有围绕三块：视频标题、视频封面、视频关键帧。

如果发确定违规，将会进行：删除视频、降权通告、封禁账号等处罚。

# 2. 账号定位

在做视频号定位和内容的时候，我们一定要做垂直细分领域，切忌什么内容都发，那样会让你的账号很乱，粉丝很杂，不利于后期的变现。就像我们要买衣服，首选服装专卖店，如果这家店衣服，鞋，首饰都有，很多人可能不去选择，因为看到的类目太多，会感觉不专业。

我们做内容也要专一，求精，而不是求多。这样我们吸引来的粉丝都是精准粉，都是有效粉丝。所以，做视频号和开一家店铺一样尽量选择一个垂直领域的方向。

# 3. 人群画像

如果我们把年龄进行交叉分析，就可以明显的发现一个规律：从 2022 年上半年起，30 岁以下的人群在新增购物者中占比越来越高，甚至 19-24 岁成为了占比最高的年龄段（2021 上半年占比高可能是因为年轻人尝鲜的目的，限于当时的基础设施不完善，并未真正形成习惯）。如果再看 45 岁以上的 “中老年”，其流入占比一直保持稳定，作为中坚力量没有太大变化 - 现在视频号正在向 “老幼通吃” 的轨道前进。

其次，31 岁 - 45 岁 “成家立业” 人群的消费能力更高。

按照上述的逻辑，每年新增消费者年轻人占比增多，中老年占比稳定，那么中间这群人是不是要被视频号 “淘汰” 了？事实上，我们认为中年人在视频号的增量不体现在 “数量”，而在于 “质量”。

调研显示，36-40 岁的人群有着最高的消费频次，甚至高于传统意义上 45 岁以上的中老年。而从单次消费金额的角度来看，31 岁 - 45 岁这个年龄段，大于 100 元的占比显著高于其他年龄段 - 而 36-45 岁的 80 后，有着最高的占比。其中，41-45 岁的人群，大于 300 元的占比显著高于其他人群。这或许说明，“中产” 年龄段的人群有潜力成为视频号的第二个中坚力量。

第三，25-40 岁更年轻人群的消费能力正在提升。

我们再把年龄段往下降 5 岁来看 - 这个人群其实重合了一部分 30 岁以下的年轻人。数据显示，对比 2022 年该年龄段有着最显著的消费频次提升，以及单次消费金额提升。如果把标准放到 “比 2022 年高很多”，那么 31-35 岁的人群无论是频次还是金额变化都是最显著的。这可能意味着，正处于事业上上升期的人群，也有着很大的消费上升空间。

综合以上观察来看，31-40 这个交叉的人群，不仅当前消费力强，而且提升大，或许值得品牌进一步挖掘和深耕。

# 4. 视频号的变现方式

# 1. 广告收益型

赚取广告费，是各大视频号最常见的变现模式。如果你的视频号有一定的粉丝粘性，可以与商家合作接视频推广赚广告费，或在视频发布文案中植入链接。主要有平台广告分成和自营广告接单。

平台分成，每个平台的标准不同，但平均下来，1 万次的阅读 / 播放差不多有 20 元的收益，如果是粉丝和新用户阅读，那收益会更高。尽管现在视频号广告分成的权益还未开放，但我们看公众号、小程序就知道，政策不会太远。

自营接单广告，如果你粉丝量大，视频播放量高，那通常你的回报也相当不错。当然，如果你是垂直的媒体，哪怕阅读量不高，就算只有 1、2 千，也有投放的价值。

# 2. 电商变现型

但凡是和消费挂点勾的媒体，都会做电商的变现。卖水果、服饰、家电、健身、美容等等产品，消费领域越垂直，变现的效率越大。那些做抖音、快手的人设号，从人设定位、内容养号再到直播卖货，底层逻辑是不变的。

所以，只要你在视频号上积累的粉丝足够多，你也可以来卖货，而且是一定可以卖出去的。无

论你有货源或是没有货源，都可以考虑电商变现模式。有货源赚利润，没货源赚佣金，目前视频号支持多家外部平台的商品分销。通过短视频分享商品，吸引用户兴趣，引导购买。

# 3. 个人 IP 变现型

如果你的视频主体拥有一定的大众认知程度，那完全可以不拘泥于一种方式。无论是出品 IP 联名款周边，或是基于用户推广辅导训练营，以及直接开启视频号直播带货，都是不错的选择。

如果你是瑜伽达人，那么你可以招募一批对瑜伽感兴趣的人。通过付费会员的方式，为他们提供个人身材管理的指导、线下活动交流、线上闭门的课程分享。其实社群会员，本质上就属于经营你自己的超级用户。

你也可以把自己对这个行业某个领域的认知进行总结，然后出自己的课程。可以是线上的直播课、也可以是系列课，还可以是线下的大课。这就是知识付费，用户是真心乐意为有价值的内容买单的，因为相比他自己琢磨划算的太多。

你还可以基于你所擅长的领域，为个人或企业提供商业方案的咨询与设计。小到个人一个小时的快速付费咨询，大到一个企业一年的商业咨询和方案策划。就像张琪的视频号，就是专门服务各企业的，提供商业课程。

# 4. 线上教育型

最早入场视频号的一批人中，线上教育机构及个人占了很大一部分。随着产品功能的迭代完备，目前线上教育在视频号的变现探索，早已有迹可循。

有人用微信小商店售卖教辅书籍，有人以免费领材料福利来吸引用户推广培训班。同时通过直播来讲解产品的优势及服务细节，更加直观的为用户打消疑虑，缩短决策时间。

# 5. 线下导流型

早有分析指出，本地生活类内容在视频号潜力极大。在线上流量向线下流量导入的过程中，为商家提供更多变现可能。

如果你是美食博主、旅游博主等，可以与本地餐饮店合作，用视频号 + 公众号为餐饮店家做推广，赚推广费。

如果你是门店人员，也可以结合门店产品卖点或服务，输出短视频 / 直播内容，引导消费者到店体验。

# 6. 私域变现型

就是把你的粉丝引流到你的微信中，在发布视频或直播时，引导添加个人微信或企业微信，先把流量掌握在手中，即便第一次不买，但吼吼就与用户构建了直接链接。同时也要注意账号的精细化运营，无死角的通过私聊，群聊或朋友圈影响客户。不过，在这里我建议不要选择私聊或者群聊。因为现在我们每个人的微信业务太繁忙了，你总是打扰别人的话会被直接拉黑。最省时省力的就是经营好你的朋友圈。如果一个人想要了解你，他第一件要做的事就是翻烂你的朋友圈，所以你要运营好朋友圈，获取信任。

# 7. 直播带货型

直播带货是视频号为创作者提供的一种通过直播形式进行商品信息分享的变现方式，通过它可以帮助你获得一定的收入。直播带货是目前最为直接的变现方式，视频号对货品的要求相对宽泛，可以是实物，也可以是课程或服务的虚拟产品。

目前的带货主播有阶段性直播的大咖，比如李筱懿。也有很多每天固定时间专职直播的主播。

# 6. 推送机制

# 1、社交推荐

触发社交推荐条件则是基于微信生态下好友点赞、评论、转发、完播。通俗来讲，只要有用户点赞，即可获得流量，不仅依赖于流量投放。所以在视频发布后我们会第一时间将视频分享至好友、微信群、朋友圈，让他们点赞。

# 2、算法推荐

通过大数据平台，根据用户行为轨迹、兴趣、视频内容偏好、职业、年龄等数据，推测出用户可能喜欢的内容。所以在创作中创作者应该多添加话题和标签，有助于被官方推荐。

# 3、兴趣推荐

当你进入视频号后，视频号默认推送推荐这一栏内容，这时官方就会根据你的兴趣偏向给你推荐你可能感兴趣的内容，例如当我们收藏了几个萌宠类视频或查看过朋友点赞过的视频后，推荐栏里推荐的都是类似内容，所以在定位视频内容时，越垂直越好，增加用户粘性。

# 4、地理位置推荐

基于 LBS 定位，在发布作品时带上地理位置标签，大几率会被推荐给同城用户，对于地标类、打卡类、探店类视频账号尤其重要。

在刚启动视频号时，利用好平台推荐机制，善用运营规则，从中获取官方流量是运营中不可或缺的环节。

# 2. 用户画像

根据抖音官方公布的数据，抖音用户年龄分布较为广泛，主要集中在 18-35 岁之间。其中，18-24 岁用户比例最高，占比为 35%，25-30 岁用户占比为 27%，31-35 岁：占比约 16%，这说明出抖音作为短视频平台在年轻人中的受欢迎程度，同时，抖音也在不断扩大其他年龄段用户的覆盖面，让更多人能够享受到其提供的内容和服务。

抖音的内容涵盖了各个领域，从美食、旅游、音乐、时尚到健身、母婴、萌宠等等，满足了不同用户的兴趣需求。根据抖音数据显示，用户最喜欢的内容是音乐相关和舞蹈，占比分别为 30% 和 21%。此外，美食、旅游、萌宠、时尚等领域也备受用户喜爱。

根据抖音数据显示，用户主要集中在一二线城市，例如北京、上海、广州、深圳等地。其中，北京、上海、广州的用户占比较高，分别为 7.9%、6.7%、6.3%。此外，江苏、浙江、河南等省份的用户也较为活跃。

# 3. 时间管理

# 抖音三个黄金时间段是什么？

** 上午 9 点至 11 点：** 这个时间段是工作日上班族使用抖音的高峰期。因为上班族一般会在上班前或中午休息时段打开手机玩抖音，所以在这个时间段发布视频可以更容易吸引到目标受众的注意力。此外，由于上午是大多数人精神状态较佳的时候，他们更容易对有趣、有创意的短视频产生共鸣，从而增加你的曝光率和粉丝数量。

** 晚上 7 点至 10 点：** 这个时间段是大部分人下班后休闲娱乐的黄金时段。人们通常在晚饭后休息时会打开手机，尤其是在周末或假期。在这个时间段发布视频，能够吸引更多的目标用户观看和互动。同时，因为晚上是人们放松心情的时间段，他们对有趣、搞笑的短视频更感兴趣，所以在这个时间段发布适合娱乐休闲的内容，可以获得更多的点赞、评论和分享。

** 深夜 12 点至凌晨 2 点：** 这个时间段是夜猫子、宅男宅女们最活跃的时刻。很多人在深夜还没入睡的时候会打开手机，在抖音上寻找娱乐。在这个时间段发布内容，可以吸引一些特定的夜猫子用户，特别是对于涉及到夜间娱乐或者消遣的话题，比如美食、游戏、恐怖片等等，都能够在这个时间段得到更多的关注和转发。

# 5. 内容定位

一、做账号前先思考

1、账号定位：要做什么、目标是什么、方向是什

2、内容定位：自己优势、擅长领域、可用资源

3、商业定位：视频带货、直播带货／打赏、金主广告

二、准备工作

1、账号层面：养号 5-7 天，浏览内容尽量与定位相关；进行点赞、关注、评论、分享；目的提高账号初始权重

2、账号设置：账号命名、定位类别、背景墙、描述、标签

3、拍摄设备：前期手机即可，有条件准备手机支架、打光灯等

4、视频处理工具：剪辑处理：快剪辑、爱剪辑②片头处理：乐秀、 Quik③声音处理：配音变声器④水印处理：水印宝、 inshot5、素材平台：创可贴（图片素材）、新片场（视频素材）6、数据平台：飞瓜数据、西瓜数据三、内容定位

1、账号目的：品牌宣传？打造网红？直播带货？等

2、定位领域：

①可选择领域：美食、美妆、搞笑、娱乐、宠物、艺术、运动、健康、情感、舞蹈、旅游、服饰、母婴、房产、汽车、科技、教育、家电家装等②不可选择领域：金融产品、医疗、保健品等

3. 账号布局

①1 对 1 强关联：多人短视频、推荐闺蜜朋友、情侣号②多号单领域：品牌号＋垂直细分领域多号多领域

4、核心价值观：有用和有趣两种①有用价值：知识类、技能类、故事类等②有趣价值：感官体验、情感满足等

5、表现形式①实拍＋字幕：真人出境、 vlog 、变装②动画＋字幕：吉祥物品牌、科幻类③其他：图文类、创意 3d

6、设置记忆点：如固定结尾、固定开场、创意动画等

7、拍摄要求：真人出镜、时长 20-30 秒、结合热点，宣传励志、正能量、美好生活内容四、常见爆款视频类型技能类：生活技能、艺术、学习、情感等高效类：内容精心打造剧情、男女反串、街头撩妹帅哥等夫妻类：宠溺类、霸气老婆、相爱相杀等晒娃类：可爱萌娃、幼师一堆娃、戏精等宠物类：卖萌打架、戏精等美食类：家庭美食技巧、创意美食、家乡特色美食等旅游类：美景、挑战极限等其他类：励志、通小故事、行业背后等美女帅哥类：歌舞才艺、换衣变装、吃喝逛街日常等
