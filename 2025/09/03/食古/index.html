# 2. 逻辑定义

# 3. 市场调查

目前市面上存在较多同类型竞品，例如《下厨房》APP，但是针对传统文化美食较少。

# 4. 基本设计方案

提供一个网页客户端，将文章与传统食物关联，可以相互检索。（建立食品文章数据库）在首页每日推送一篇文章和相关联做法。

吃点古早味儿

# 7. 需求定义

  • 提供菜品的具体制作方法，包括菜品食材，要点和操作步骤

  • 提供传统食品文章

  • 提供食品检索方案（按照食材、厨具、名称等方案搜菜）

  • 提供用户自定义发布探究古食品文章的社区

  • 提供本地笔记与记录系统

# 8. 针对人群

不知道吃什么的年轻打工人，喜欢中式传统美食的人
