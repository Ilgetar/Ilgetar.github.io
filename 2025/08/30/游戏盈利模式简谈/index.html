游戏付费模式浅谈

# 游戏付费起点

游戏是动物的天性。

世界上第一款电子游戏是《双人网球》。关于世界上第一款电子游戏有很多种说法和版本，包括 1912 年的国际象棋机器 El Ajedrecista、1947 年的第一款电子视觉化显示器阴极射线管娱乐装置、1951 年最早的数字计算机游戏机 Nim、1952 年第一款电子显示器显示画面的井字棋游戏 OXO 等，主要分歧点还是在于对 “电子游戏” 的定义上。这里采用的是时间上相对最晚的一个版本，因为双人网球有实时运动的画面。在早期的游戏开发中，游戏本身大多数是用于研究人机交互的产物，到后面才慢慢进入商业领域。

在 1971 年发行的《电脑太空战》，被认为是第一款商业街机游戏。1972 年创建了 Atari 公司，开始发行街机游戏。但是这款游戏并不受欢迎（因为玩法太复杂了），后来就发行了《PONG》这一款乒乓球游戏机，正式进入市场。随着时间发展，各类游戏主机也逐渐进入中国，直到 1991 年开始风行全国。

# 游戏付费种类

# 买断式付费

玩家需要支付游玩游戏本身所需要的费用，一次性支付，永久拥有游戏游玩权。这种付费模式是最原始也是最早期的游戏付费，直到今天也还是有许多厂商选择这种付费模式，当然有些游戏厂商也会结合其他的付费模式，例如《刺客信条》就在买断付费的条件下提供了游戏内部的道具选购，玩家可以通过二次付费提高自己的游戏体验。我们现在常见的独立游戏，例如《空洞骑士》《双人成行》《黑神话：悟空》等都是独立的买断制游戏。这类游戏的最大特点就是游戏流程较长且较为固定，游戏体量较大，开发存在较大的门槛，需要直接对游戏本身付费。

这类游戏的一大问题就是，一款游戏如果你不亲自上手去游玩体验，你很难知道游戏是否适合自己，所以玩家出于顾虑也就少了很多冲动消费的机会。游戏本身也会由于体量和制作成本的原因，导致游戏本身的费用较高，准入门槛较高，游戏受众就没有那么广。当然后来同样的也就出现了退款和先行体验版等解决模式。

# 游戏内购付费

玩家可以免费游玩游戏本体，在游玩过程中为玩家提供一些游戏内资源的付费选项。基本上游戏内的大部分东西都可以通过这种内购模式来获得，包括资源、角色、装扮等。比较经典的有：《原神》《明日方舟》《碧蓝航线》《第五人格》等。这类游戏的特点就是长效运营，官方会持续不断的产出新的游戏内容，常玩常新，同样的也会提供越来越多的付费点，更优质的内容吸引玩家付费。

这种模式有一个最大的优点就是付费门槛低，很多游戏最低可以只充值 1 元或者 6 元，降低了付费的门槛，也就更容易有更多的玩家付费。并且由于游戏本身的长期运营，这种付费模式可以源源不断的为厂商提供资金收益，玩家也很容易不知不觉中在游戏内付出更多的资金，相对于传统的付费模式，厂家的收益大大增加了。

# 游戏付费游玩

玩家需要付费才能进行游戏的游玩，一般按照游玩时间计费。从早期的按月度收费，到后面出现的点卡制度，都是付费进行游戏游玩的模式。这种游戏模式是对于买断制游戏的补充，让玩家可以有尝试的倾向，也提供了及时止损的机会。与此同时，厂商也可以获得源源不断的游戏资金以支持游戏后续开发维护。

# 游戏外付费

通过游戏外的 IP 开发、赛事、周边众筹等形式提供游戏所需要的资金。这一类付费模式在当代游戏中多有体现。相对于虚拟的游戏数据，玩家们对于游戏内的喜欢也会映射到现实世界中去，采购自己喜欢的角色立牌周边，观看游戏对应的赛事，去参加相关的活动。这种付费也是对游戏本身的扩展和延续，可以为很多游戏的重负玩家提供更多的付费选择。这一块比较常见的比如《王者荣耀》赛事，《第五人格》赛事，《明日方舟》周边设计，都是可圈可点的案例。

# 非游戏付费

主要通过联动、广告等信息提供的付费选项。厂商可以在游戏中添加游戏相关的 IP 联动、广告等信息获得收益。比如《咸鱼之王》的广告、《喵斯快跑》的联动。大多数的微信小游戏都会在游戏中添加广告、添加游戏内道具付费来获得收益。这一部分的付费模式就适合小体量的游戏，玩家通常是为了打发零散的时间，并没有追求进一步的深入游戏体验，只是图一个新鲜，所以付费模式更加追求即时性、可靠性。

# 玩家付费选择

在不同的情况下，不同的游戏群体会对同一个游戏的付费模式和内容产生不同的选择和评价。现在的市场上，每天都有大量的游戏在不断的生长和消失。每一种游戏付费模式都有适合的对应人群。学生党会关注高质量的买断制游戏和免费的长效运营内购游戏，社会职场人士可能会关注怀旧版时长付费和及时付费小游戏以缓解个人的生活压力。

# 付费模式的方向

个人认为，各种游戏付费的模式都有其特点，游戏是什么样子的，就会有对应的游戏付费机制。但是到最后的最后，还是会回到游戏一开始的目的：娱乐。玩家选择游玩一款游戏，一定是出于自己的一个情绪在驱动。要影响到玩家的意愿，终点一定是内容为王。这就和媒体时代的结论一样。你开发更多的游戏付费，推出各种各样的收费形式和收费项目，本质上都要建立在玩家喜欢和意愿的前提之下。玩家发现付费获得的游戏资源并不够优质，付费并不能获得更好的游戏体验，那么就会变得愤怒，并且退坑游戏了。在这一款我必须要提到两款游戏，一个是《碧蓝航线》，一个是《尘白禁区》，两个游戏都是一开始近乎死亡的起点，但是随着内容的不断扩充和纯净的消费驱动，后面才能蒸蒸日上。比如说《尘白禁区》，基本上所有的角色和装备素材都是可以通过免费获得，游戏本身的难度也不会随着充值的程度有改变，基本上就是不充值就完全能爽玩。但是玩家会为了一个优质的外观付费，并且大量付费，选购喜欢的角色周边，反而在游戏里投入相对于其他游戏更多的资金。

说到底，玩家所求的就是一个心理体验。花钱买的是服务，这就是游戏付费的终点站。在服务行业，顾客就是上帝。总是对着玩家颐指气使，终将失去讨论的资格。

游戏付费 杂谈
