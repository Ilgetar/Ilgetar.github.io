# 1，认识论问题：纯粹理性批判

背景：感性 VS 理性

理性主义认为知识是先天的

经验主义认为知识是后天的，经验的（休谟走到极端则认为一切需要经验检验）

康德一开始是理想主义，后来直面休谟的问题，写出纯粹理性批判，目的：先天综合判断如何可能？

知识：判断（什么是什么）

先天分析判断与后天综合判断

例如：“红花是红的” 必然性，先天判断，不产生新知识

“红花是香的” 经验得出，没有必然性，能产生新知识

康德提出先天综合判断：7➕5＝12 是一个综合判断，但是具有必然性

认识论的哥白尼革命

传统认为，自然是有规律的，主体就是在认识规律。康德认为是自然符合了我们的先天认知形式。

（1）先验认识形式：时间与空间

是人在为自然立法。

（2）物自体

（3）二律背反

例如宇宙是不是无限大

康德提出，必须要为理性划清界限，人只能认识到人能够认识到的事物

（4）感性 / 知性 / 理性

从感性出发，通过认识能力，到理性的形成

# 2，伦理学（实践理性批判）

前提：人是自由的。人具有自己选择的能力。既然这样，人就有自己的原则和理性。

康德认为道德来自于理性。

（1. 人为自己立法：所有的道德都是人为自己建立的。

  • 假言命令：有条件的（赚钱了就讲诚信）

  • 定言命令：无条件的（不管怎么样都讲诚信）

在康德认为，只有定言命令才是道德。而且道德必须是可普遍化的（其中可普遍化源于信仰）。

道德一定是个体普遍化之后的结果。例如借钱不还可以是个人的原则，但是不能普遍化，所以这不是一个道德。

（2）人是目的，不是手段。（例如电车难题）

（3）必然与自由的矛盾

处于现象界的人：必然的

处于本体世界的人：自由

纯粹理性批判实践理性批判
认识论问题道德问题（伦理学）
人有感性➕知性＝概念 / 范畴＝＞理性人是自由的➕理性＝＞可普遍化原则＝＞道德

两者之间连接的就是判断力

# 3. 美学（判断力批判）

判断力：将知性与欲求结合的能力

康德对于审美判断提出四个契机（关键）：

  • 无利害而生愉快

美感和快感：快感使人愉悦，美感一定让人愉悦，快感不一定是美感，例如夏天的冰棍，并且快感具有功利性。

  • 无概念而具有普遍性

无概念：认识论范畴，概念判断不具有审美形式。但是概念有普遍性的，审美虽然没有概念，但是依旧有普遍性：因为其无功利性，所以所有人都应该会觉得美。（寻求普遍性）

判断，愉快，谁先谁后？康德：. 一般愉快是先感到愉快，再生判断。而美感是先有判断，才产生愉快。

  • 无目的而合目的

一般传统目的是经验目的，而康德认为有一个先验目的（人先天有一个目的，与心意状态一致）

即本来无目的，但是后来符合了先验目的，所以我判断符合了，才感到愉悦和美。例如理想型女孩，先判断符合理想型，才觉得美。

康德认为美分为两者：纯粹美与依存美（目的）。主要探讨纯粹美（形式上，无目的），例如大自然，纯形式

  • 审美共同感

例如审美的寻求普遍性，认为人一定是存在某种共性。

自康德后的美学研究就从美的本质问题走向了心理学（美感论，精神分析，如弗洛伊德）和社会学方向（审美建构论，如马克思）。也是从康德开始，美的本质问题走向了解构。

# 第十二讲 艺术哲学②：艺术批评与审美趣味问题

艺术阐释、艺术批评、艺术评论。目前没有一个统一权威的流派。

路易斯 - 布尔乔亚《母亲》蜘蛛像一种无休止的劳作，一直在修补。

作者的创作意图能不能成为我们阐释艺术作品的关键？

# 一、阐释意图：意图主义与反意图主义。

意图主义：作者的意图决定了作品的全部意义。

前提：艺术本质（模仿与表现）

如果你坚持意图主义，则你默认了艺术本质是一种表现。

# （一）意图主义（预设了表现论）

1. 偏重直觉的，直觉表现论：克洛西

艺术家在创作之前，不知道自己要表现什么

Q: 无法追问作者意图，只能回到作品本身，猜（领悟）意图

2. 偏重交流的，交流表现论

创作之前，作家已经知道自己要表现什么。

Q：在阐释批评的时候，要绕过作品，直接追问作家意图。那么关于作家的材料就会重要。艺术作品已经完全不重要了。

# （二）反意图主义（新批评）

艺术家的意图不重要，首先要看作品！

1946 年，比尔兹利、维姆萨特《意图谬误》：追问作者意图的行为是有谬误的。

1. 艺术家是人，艺术品是物，二者不等同。意图是私人的，作品是公共的，现在你把私人的东西作为了公共的。二者是有区别的，不能混为一谈。

2. 艺术品的 “说话者” 不是艺术家 本人，而是艺术家的某种假装。

3. 艺术品的意义不是由作家决定的，而是由特定的艺术规则决定的。

Q：很容易走向虚无主义。容易过度阐释

# （三）新意图主义

赖阿斯：作品的意义≠他的效力，作品的意义应该是由作品推断出来的意义。这种就是作品意图。

巴克森德尔：《意图的模式》，意图这个概念应该扩大。传统的就是作者的心理，现在应该包括作者的社会历史背景。

沃尔海姆：回溯式批评：艺术阐释的时候其实就是一种创作过程的回溯。

艺术价值到底源于哪里？

# 二、评价对象：艺术价值是主观的 / 客观的

# 1. 主观主义

艺术品的艺术价值存在于评价者的主观喜好。那么每个人的观点都是真的。将会导致无意义。进而产生相对主义。

# 2. 客观主义

艺术品的艺术价值在于其本身，不以人的意志而转移。如果你的评价能够反映艺术价值，那么这就是一种真的艺术评价。

大多数认为，客观性来源于历史性（社会）

英国沃尔海姆《艺术及其对象》：

# 艺术价值的承担：

艺术本身；艺术品；艺术特征

# 艺术价值的断定：

①现实主义（real）：艺术品的审美价值具有真理性，不以人的意志转移。具有原初性。

②客观主义：艺术作品的价值仅仅在审美问题上独立于人，不以人为转移。具有第二性。

③相对主义：艺术作品的审美价值依赖于人的心理，还与人群有关。不具有普遍性。

④主观主义：艺术作品完全具有一种表现性特征。

# 三、阐释结果：一元论与多元论

一元论：在众多阐释结果中必定存在一个唯一的最好的结果。

多元论：允许各种解释存在，所有的个体解释都是合理的。

# 四、批评标准：审美趣味问题

审美趣味作为一种能力，其实应该能评价高低，但是审美没有办法可以测定，无法做评价。审美具有私密性，但是又有普遍性。

（一）休谟

《论审美趣味标准》

1. 审美趣味标准的必要性：

审美标准必须建立在普遍人性之上。

2. 趣味高低如何判断？

文艺复兴塞万提斯的《堂吉诃德》：桑丘评酒。

其实人关于味觉是具有不同的敏感力的。能区分细微成分的人就是厉害的。

那么决定审美趣味高低的就是审美敏感力。具有优秀审美敏感力的人，就有标准的决定权。进而审美是一种能力，那么审美能力也可以提升。

以上思路存在循环论证，还有精英主义，以及偷换问题。

科恩：如果存在好坏的审美趣味，那我为什么需要好的审美趣味呢？酒神精神吗？

艺术品与生活用品无区别。当今时代艺术不再是传统艺术，而是后艺术的天下。

# 第十四讲 艺术哲学④ 日常生活审美化问题

审美本该是非功利性的。但是当代不是，例如购买杯子，不仅仅是个审美活动，还有功利性。

# 一、日常生活审美化与时尚

影响：

高雅文化与大众文化的界限模糊化。

对身体的修饰（身体工业 / 美丽工业）

总而言之，导致人们注重生活经验，让人感受到一种自由和平衡感。

在传统美学看来，所谓的感性是和理性对立的，而且理性地位更高。而在当代问题中，日常生活审美化在让感性独立出来。

符号消费本质就是在消费审美。
