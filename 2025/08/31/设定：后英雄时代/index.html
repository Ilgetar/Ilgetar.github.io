# 人物设定

# 余裁

非工作情况通常喜欢保持沉默，情绪稳定。理想主义者也是悲观主义者。前文明的遗民，体能一般，但是思维敏捷，善于说谎。旧时代先锋科研队成员，在英雄时代苏醒，隐姓埋名却又重新成为了记者。在冷冻柜里醒过来的古代人。在新时代苟且过活，隐姓埋名不谈论旧日文明的秘密。

熟悉多国语言和古语言，擅长绘画和摄影，喜欢音乐和游戏。擅长倾听，对于语言有自己的理解和构想。

通常携带一副头戴耳机和一台微单相机，背包里面装配有常见的工作设备和特殊设备（存疑）。喜欢身着褐色大衣，黑色眼睛灰白头发。出生于旧历 239 年，于 262 年冷冻，在元历 101 年被解救。（一共冷冻了 111 年），皮肤白皙，喜欢昼伏夜出。

A107 年时 29 岁。期望：回到故乡。

“这里的一切都让我感到作呕，可怖的琐碎和嘈杂的人心。我想要回到故乡，哪怕牺牲这里的一切。”

# 吴峰

英雄时代的第一代先锋，和余裁是好朋友。后来因为不知名的原因失踪。

A107 年时 28 岁。期望：解开一切的真相。

“我只想活在这里，此时此刻，有你有我。可是无知造就毁灭，要改变未来，我必须前进。”

# 肖莹芙

余裁前女友，情绪不稳定，财阀独女，样貌可人。

A107 年时 24 岁。期望：维持更稳定的地下城。

“明明大家在地下活的好好的，你们又为什么要毁掉这一切？我们完全可以自给自足的。我不想看到任何伤亡。”

# 刘加

英雄委员会第一任主席，后退居二线。外在开朗友善，实际城府很深，为了目的不择手段，有自己的道德尺度。妻子早早去世，非常思恋。

A107 年时 41 岁。期望：复活爱人。

“我爱她的乐观，她的睿智，她的温柔。我想要再见她一面。”

# 艾伊莎

旧日人造人，同余裁同年苏醒。目前处于失踪状态。

A107 年使用年限 17 年，剩余 3 年。

“RQ-3584 号，你的责任是如实记录渊默文明的一切。”

# 时间设定

旧历 269 年，爆发大静谧灾变。旧时代人类几乎全灭。三年后，剩余的人类重建了少数的地下城文明。于旧历 272 年启用新历，定该年为元历 1 年。

元历 76 年，爆发次级入侵，人们被迫面对未知的虫群，死伤惨重。

元历 78 年，重新组建英雄队伍，开启英雄时代。

元历 101 年，英雄队解救余裁。

元历 106 年，开启后英雄时代。

# 主线规划

故事从元历 107 年开始，余裁开始调查后英雄时代的真相。逐步发掘旧日的秘密。

渊默文明是一个纯净的文明，追求真理与发展。但是理想国容不下污浊与背叛。在绝对的 “理智” 之下，使用随机算法选择淘汰了一半以上的生命。在地表投放了大静谧病毒，但是有少数旧人类活了下来，悄悄在地底建立了地下城。并且销声匿迹。

后来渊默岛发现了地下城的存在，设计了次级入侵的机械虫群，再次重创地下城。但是此刻渊默岛的旧人类发现了内鬼，出现了分歧，人的纯粹性被彻底瓦解，出现新宗教。最后渊默岛被摧毁，幸存的岛民流落地表，很多也不再幸存。后来勉强建立了新的地表文明，重新开启文明的发展。

# 思想设计

1. 宗教总是在试图摧毁你的心智

“请把你的鞋子和头脑留在门外” 一旦有了强制力，就会有机会作恶。那是一种傲慢且忘乎所以的欲望，想要在朝偶像跪拜时页摧毁心智。

我们天生不会渴望一种朦胧的迷幻的无忧之境，而把批判和反讽的能力作废。

2. 永远留意语言。

3. 语言永远无法达到纯粹的真实世界。
