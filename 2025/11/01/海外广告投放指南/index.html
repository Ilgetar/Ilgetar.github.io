# 写在前面

​ 这里的笔记是关于【海外广告投放】的，来源是我之前求职参加的一个入职培训，不过很遗憾的是，那次培训的通过率只有可怜的 30%，并且培训的补偿也只有 30 块一天。不过说起来，对于刚入社会的普通学生来说，这样已经很好了。所以在最后，我只能以一分的差距遗憾离场。不过在那里了解学习的东西，我也觉得多少有点作用，就在这里整理出来吧，好说也是行业内入行需要的基本知识。

# 行业现状

​ 目前主流的海外四大媒体有：Facebook，google，Tik Tok 和 Kwai。其中 Facebook 应该是最早的付费广告开发商，不过后来被其他的流媒体平台有所追上。从国内来看，主流的广告投放也在不断向短视频等流媒体平台倾斜，以后短视频等形态的广告投放一定会成为主流（当然传统媒体广告也不会被淘汰）。因为视频形式的信息传达效率就是会比传统图文形式更加高效和诱人，如果说以后出现 3D 的 VR 广告形式，也会在基础设备完善之后高歌猛进的。

​ Facebook 是全球最大的社交媒体，不过在国内无法使用，需要科学上网。在 2021 年 Facebook 公司改名为了 Meta，寓意是元宇宙（从这里可以看到他们公司的愿景和野心，但是很遗憾只有名字的改动）。Facebook 其上的广告是一种付费的广告服务，它允许广告主通过自选的形式向特定的用户和受众进行广告的展示。这有点像国内的微信，你可以通过花钱来购买朋友圈或者视频号的广告位，然后还可以筛选广告的投放人群。他们家的广告主要有四种形式：图片形式、视频广告、轮播广告和精品栏广告。每一种广告有其不同的广告价格和规则，比如说，图片广告的文本含量不能超过该图片的 20%，如果放置在桌面端的右边栏，则标题不能超过 25 字符，详情不能超过 30 字符；快拍里面展示的广告图片每一张只显示 5s；要在即阅文里面展示的图片广告只会在移动端投放。

​ Google 是全球最大的搜索引擎，也就是说可以在网页投放对应的广告。当用户进行某些特定词条的搜索的时候，搜索结果就可以展示这份广告。当然这个广告会明确标注出【广告】的标签，这和国内的百度等引擎是一样的，只不过因为百度的广告实在太多了，现在被骂的很惨，后续也就涌现了越来越多的搜索引擎。对于在网页端展示的广告，其实可以通过不付费的方式进行投流，也就是使用 SEO 技术，对于网页的一些关键词进行优化，尽可能钻系统的空子，然后让自己的站点显示在搜索结果的前列。如果使用付费的方式则称为 SEM 投放，也就是上述的广告投放模式。

​ Tik Tok 实际上是国内抖音的出海产品，现在是国外非常有影响力的短视频社交平台，遍布 150 余个国家和地区，涵盖近 35 种语言。在短视频上面的广告投放其实就是我们可以刷到的信息流广告，你刷着视频，突然刷出来一个广告，或者某个视频的内容本身就是广告，然后在便捷的地方提供了购买的链接。这其实也是我们国内电商的运转模式，电商公司可以选择联系一些头部的优秀博主，让他们代言推销自己的产品，比如说李某琪；或者公司自己拍摄大量的广告素材，然后在抖音提供的广告管理工具【巨量千川】里对自己的付费广告进行管理和优化。国内大部分的电商公司基本就在做这种事情，及时优化自己的广告视频的数据，及时管理互动评论，刷评论刷点赞，尽可能提高自己广告的 ROI（投入产出比）。

​ Kwai 则是相对于抖音来说类似的短视频平台，不过 Kwai 更多的是针对拉美、中东、东南亚等地区，更加关注下沉市场。抖音相对来说就会在发达国家更加流行一些，某些地方甚至会影响到政治选举，然后狠狠被制裁。

​ 如果你要在国外进行产品的销售和推广，一般来说要有自己的品牌独立站，相较于国内来说，外国人在选购商品的时候还是很看重你的品牌的，会访问你的官网。那么这里就会有各种各样的需求，比如说要建立站点，完善商户信息，采购国外资源，进行广告信息的优化，选择推流方式等等，这些都是需要大量的积累和工作量的，有需求就会有市场，于是乎就出现了很多专门做品牌出海服务的公司。甲方会把相关的需求提交给这些企业公司，然后公司进行后续的各种操作和服务，就像是一个代理。

​ 你也许会问，这些活听起来也不难，什么买电话卡，创建广告账户，填信息，那些公司不能自己做吗？当然可以，但是一个不是专门做这些工作的公司，他们来做会遇到各种各样的问题，工作量也大，还需要试错，比如说电话卡被封了怎么办？广告效益低怎么优化？这些都是需要积累和经验的。这时候花钱解决问题才是最划算的。专业的事情交给专业的人来做，这就是以后的发展趋势。

​ 听培训老师的介绍，这个行业可以挣到很多的钱，因为甲方提供的利润空间还是很大的，有时候广告投放的预算是非常多的，你的效益好，提成什么的就会很好，一个月甚至能到几万的工资。不过因为我没能留下来上班，这些话不保真哈。不过照我看来，广告行业确实一直能活下去，挣多挣少就看你个人了。

# 投放模式

# Facebook 引流

​ 在脸书上，引流主要有三种模式：公共主页、付费广告和直播引流。

​ 公共主页是企业对自己客群的展示页面，也叫做” 粉丝主页 “。一个公共主页只能唯一对应一个广告账户，但是一个个人账号可以对应多个公共主页。也就是说，公共主页就像是国内的公众号，可以有很多个人去管理这个公众号，但是公众号只能对应一个充值的广告账户。同样的，只有公共主页可以做商业的推广，如果一个个人账户进行商业推广，是会被封禁的。在公共主页里面，商家和品牌可以及时更新自己的咨询消息，传达给关注的客户群体。需要注意的是，个人账号也会有一个对应的个人主页，并且也不能做商业推广。

​ 付费广告就像上述所说，是广告主通过付费，来选择自己所需的广告受众，以便于更好的广告投放效果。特点就是可以短时间快速起量，缺点是需要专业的人来操作，有一定的学习门槛，比较适合品牌的前期和中期，能更好的达成业务目标。

​ 直播这一块不太了解，只知道有。这里不多讨论。

# 营销漏斗

​ 营销漏斗是对广告营销的全流程中进行分析的一款模型，它一共分为【展示、点击、到达、浏览、咨询、销售】六个步骤，分别对应认知、考虑和决策阶段。在不同的阶段也就有着不同的营销目标。比如在” 认知阶段 “，需要关注覆盖人数、品牌知名度、视频观看量、流量、互动等数据；在考虑阶段，需要关注流量、互动、销量等信息；在决策阶段，就会着重关注销量这个数据。

​ 这里提供的一个由上而下的倒三角营销模型只是一个在营销的过程中的理论参考模型，在遇到问题的时候或许可以更多的参考自己的理解，要多想，只要是有用的，那就会是对的。

# FaceBook 广告竞价原则

​ 一句话来说，就是最大化广告主与用户所得的价值。需要注意的是，这里的具体计算应该还是黑盒状态，没有办法完全明晰。总的来说，系统会根据广告主给出的广告价格，计算总价值，用这个总价值和其他广告主的总价值进行竞争，价高者得曝光机会。

​ 广告主竞价 X 预估操作率 + 用户价值 = 总价值

​ 这里的预估操作率其实是指你的广告目标所能达成的可能性，由于翻译的问题可能出现误解。

# 应用类广告投放流程

  1. 利用公共主页推广应用
  2. 借助 FacebookSDK 工具对用户客群进行分析理解
  3. 针对客群特点，通过移动应用广告获得更多的安装数量和互动量
  4. 评估广告投放的成效并且总结成功和失败的经验

商务管理平台

这是一个提供给广告主使用的管理工具，具备了包括投放和追踪广告、管理 Facebook 资产、使用 BM 绑定公共主页、广告账户、像素等资产的功能。

像素

即 Facebook Pixel，一个给 Facebook 账户运营提供的一个分析工具。

主要的功能有：1. 追踪数据；2. 竞价和优化；3. 定位自定义受众。
