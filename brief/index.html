# 个人简介

​ 你好，我的朋友，很高兴认识你。我来自湖南邵东的一个小县城，03 年的水瓶座，E 人，过着和大多数人差不多的童年和学生时代，一样的暗恋，一样的叛逆，一样的愤怒和一样的沉沦。顺着理所应当的学习路径，考上了一个普通的大学，步入社会。我的脑海里面总是会冒出一些似乎不属于我自己的思想，一些晦涩的困惑和远大的梦想。我曾经也憧憬着改变世界，为天下立命，我曾经也沉迷于虚拟世界，也曾日复一日的搬着赛博电子砖，只为了那一块生活的面包。我必须要知道，人到底活着为了什么？在向死而生的道路上我们越走越远，回头看才发现早已经找不到来时路了。忙忙碌碌，碌碌无为。

​ 而我现在能想给出的答案是：为了未知。我不是一个伟大的人，我也很难成为一个伟大的人，但是我会一直为了未知而求索下去：未知的体验、未知的难题、未知的领域…… 好奇永远是我前进的动力，哪怕它看起来微不足道。这就是我二十多年来最大的感悟，是好奇这两个字推着我前进。

# 兴趣爱好

  • 喜欢阅读，但是只看自己感兴趣的东西。最喜欢的作者是加西亚 - 马尔克斯，最喜欢的诗人是博尔赫斯。
  • 喜欢音乐，最喜欢的歌手是 REOL，最爱的歌是 **《No title》**，在这首歌里我感受到生生不息的力量。
  • 喜欢游戏，市面上大部分游戏都会涉猎一点，比如明日方舟、三角洲、原神、第五人格、空洞骑士、刺客信条、喵斯快跑等等。玩的最久的游戏是碧蓝航线，在线时长最多的游戏是第五人格，不喜欢 FPS 和 MOBA 游戏，因为很菜。
  • 喜欢画画，但是很少拿起画笔，技术日渐变拉中。
  • 喜欢吃谷，是完全奔着二次元角色的美貌去的，真的好美。
  • 喜欢手工，会做流麻、抛光、折纸、雕刻、拼豆和玻璃画。
  • 喜欢摄影，但是不喜欢修图。现在用的是松下 G95D，有钱了会再升级。
  • 喜欢喝茶，是学校茶艺队副队长，喝的最多的是正岩肉桂。

# 游戏经历

忠实的游戏追求者，从远古时期的捕鱼达人和 4399 开始，一直活到今日的二游大乱斗。死去的游戏和长青树数不胜数，故事也在一天天延续。个人游戏简评：

  • 《碧蓝航线》：开服累计登录 2600 天的老玩家。持续的运营建立在三个基础之上：一是听劝，二是福利和奖励足够优秀，三是游戏定位明确不暴雷。作为一款即时海战弹幕设计游戏，游戏机制本身的可玩性无限接近于无，所有的养成和玩法都极度轻量化，又或者可以实现自动化。作为一款二游，在 8 年前就已经完全实现了通过角色外观创收的营收模式，对玩家宽容和尊重，已经是万里挑一的游戏了。
  • 《明日方舟》：2019 年开服就入坑的老玩家，直到今天依旧每天在线。很高兴制作人海猫是美术出身，并且明确的理解游戏的方向和定位。作为策略塔防类型的游戏，已经有太多的前辈穷尽了玩法可能性。我个人觉得游戏要活到今天，主要有几个原因：1，运气好，好多经典的破圈热度都是大风刮来的，比如说两面包夹芝士、出租屋里的蟑螂等爆火的梗。2，独特的游戏模式和美术风格，策略游戏经典的高拓展性和挑战性提供给了玩家极高的挑战性和话题度，别具特色的机能末世风也为普通玩家提供了关注点。3，公司本身对内容的精益求精和对玩家的尊重，公司对音乐和剧情的创作都是高标准高产出，举办的大型游戏音乐会，公益联动都是对玩家的一种回馈和感恩。
  • 《重返未来：1999》：很高兴能陪深蓝到两周年。游戏对于卡牌回合对战模式的理解简直是一场灾难，角色数值会在每一个玩家醒来之后偷偷的给玩家一个惊喜。游戏能活下来全靠偶尔超神的编剧和总是骑着策划上班的美工。9 的英伦美术设定风格，在各种细节上都是极为精致的，主线剧情里也会融入《洞穴的囚徒》，《酒神颂》等经典议题。相比较游戏，1999 更像是一本古朴的书，或者一部优雅的电影。真正的第九艺术，内容型艺术。
  • 《尘白禁区》：曾经无限接近死亡，但是在谷底突然理解了生存的真谛。曾经制作人说，要做一款最为硬核的末世射击游戏，濒死才明白，有人买账的游戏才配得上叫做游戏，不然都是路边一条。从那以后就开始了福利的全部开放，角色设计越来越大胆，直播听取玩家的合理建议，为玩家提供最好的福利和待遇，解决玩家的问题。你可以说游戏靠着擦边活下来实在不齿，但是玩家想要的不仅仅是擦边，更想要的游戏制作的态度。
  • 《第五人格》：累计在线游玩时长 7000 小时。非对称竞技游戏，国内独一档的画风和机制。有人质疑这是先上车后补票，但是游戏玩家不在意。玩家想要的是游戏本身。实际上在今天这款游戏都没有实质上的竞品，并且靠着饭圈化的运营和赛事不断创造新的利润。我至今感慨桃晚安拼尽全力抗压最后却差天半子含泪离开。电子竞技就是不断突破自我，挑战自己的过程，是人生，是苦难，也是祝福。
  • 《绝区零》：第一次公测无限接近死亡。后来制作组痛定思痛，推倒重来，给到了玩家最大的尊重和福利。动作类游戏其实比较硬核不一定适合所有人，所以制作组愿意把游戏的操作在一定程度上简化，照顾了普通玩家，也保留了高操作的可能性。对角色性格和剧情的塑造可以说是目前最喜欢，如果角色真的嗑到我的心上了，我的钱包肯定也很难守住了。
  • 《喵斯快跑》：独特的画风和休闲的玩法。偶尔玩一把是一种消遣的好时候。相对于其他音游的多音轨，喵斯的反馈和操作做的更好，简单易上手，画面丰富，是一种很简单纯粹的体验。
  • 《原神》：国内经典大世界 IP，但是过于在乎金钱的运营导致乐子频出。游戏本身内嵌了多种玩法，部分剧情也塑造的有深度有理解，没有复杂的人际 PVP，有的是二次元的岁月静好。时间长了慢慢变得冗长又重复，，又肝又氪的形式在时间长了之后就难免味同嚼蜡了。
  • 《鸣潮》：大世界后起之秀，流畅的大世界体验和丰富的福利给游戏带来了很多机会，难免会和原神做横向对比。我的评价是，相似却不相同，难得糊涂。
  • 《物华弥新》：文物主题的战棋策略游戏。二游的经典风格和新颖的文化主题是游戏的最大亮点，但是游戏设计和运营的策略已然变成了一场灾难。谁能想到在开服不到一个月的时间里，游戏战斗的数值就已经变成了纯粹的打年糕磨时间呢？
  • 《Phigros》：多判定音游，最喜欢里面的忘忧宫的曲子，有一种重拾希望的美。单机，免费，有趣，高上限。
  • 《元气骑士》：像素地牢 RPG 游戏，简单纯粹的肉鸽游戏，战斗的手感很好，道具和内容的可拓展性很高，玩法出色，可以给玩家提供更多的可能性。
  • 《刺客信条：幻景》：基于宏大世界背景下的超大型历史故事，听说里面的建筑都是有实际参考的建模。哪怕不去维护正义，也可以在街头慢慢踱步，体验一下异域风情。
  • 《死亡搁浅》：最强独立游戏，音乐和景色的配合是最纯粹的艺术品。下完雨走在山间的小溪里，突然响起来的背景音乐给人恍如隔世的梦幻感。
  • 《底特律：变人》：讨论哲学议题的剧情向选择游戏。优秀的剧情演绎是游玩的最大关注点，角色塑造优秀，剧情可讨论度高。
  • 《黑神话：悟空》：国产 3A 大作，结合中国经典文化 IP，做出了很多独属于中国的文化韵味，值得一品。
  • 《女神异闻录 5》：经典游戏 IP，体验一个高中少年的正义和救赎。
  • 《人类一败涂地》：如果有朋友一起玩，会是一种极度开心的体验，游戏本身的荒诞感和可拓展性极高。
  • 《围攻》：门槛极高的自由关卡游戏，需要对机械组装和物理效应有一定理解的思考类策略闯关游戏。
  • 《城市天际线》：在游戏中创建一个属于你自己的城市。可以参考很久以前爆火的经典：《模拟城市》。
  • 《空洞骑士》：极其优秀的打击感和极富挑战的类银河恶魔城游戏。
  • 《Journey》：探寻自我的旅行模拟器。
  • 《奥日与萤火之森》：和空洞骑士很像，但是主线更加明晰，色彩更加丰富美丽。
  • 《Stray》：猫猫模拟器，在解密中体验一只猫的感受。
